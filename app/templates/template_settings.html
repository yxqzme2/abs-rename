| SEPARATOR — custom filename
By default the last folder segment is also used as the filename. Use | to set a different filename:
{author}/{series}/Book {series_index:02d}|{series} Book {series_index:02d}
→ Patrick Rothfuss/The Kingkiller Chronicle/Book 01/The Kingkiller Chronicle Book 01.m4b

Fallbacks: no series → "Standalone" folder. No author → "Unknown Author". No title → "Unknown Title". If a segment token is missing, it is removed gracefully.
